## 매칭 조건

### 1단계: 모든 조건 만족 (최우선)

- 취미: 취미가 일치해야 함 (최소 1개 이상 공통)
- 지역: 같은 지역이어야 함
- 나이: 나이 차이가 ±5세 이내
- 성별: 성별이 달라야 함 (남자-여자)

### 2단계: 취미 조건 완화

- 취미: 조건 제거 (취미 불일치 허용)
- 지역: 같은 지역이어야 함
- 나이: 나이 차이가 ±5세 이내
- 성별: 성별이 달라야 함 (남자-여자)

### 3단계: 지역 조건 완화

- 취미: 조건 제거
- 지역: 조건 제거 (다른 지역 허용)
- 나이: 나이 차이가 ±5세 이내
- 성별: 성별이 달라야 함 (남자-여자)

### 4단계: 나이 조건 완화

- 취미: 조건 제거
- 지역: 조건 제거
- 나이: 조건 제거 (나이 차이 무제한)
- 성별: 성별이 달라야 함 (남자-여자)

### 5단계: 성별 조건 완화 (최후)

- 취미: 조건 제거
- 지역: 조건 제거
- 나이: 조건 제거
- 성별: 조건 제거 (같은 성별도 허용)
